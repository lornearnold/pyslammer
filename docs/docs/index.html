pySLAMMER

Welcome to pySLAMMER’s documentation

pySLAMMER (Python package for Seismic Landslide Movement Modeled using Earthquake Records) is tool for estimating the co-seismic displacements of landslides with rigid and flexible sliding-block analyses. The package name and code are based on the USGS tool SLAMMER by R. Jibson et al. (2013).

pySLAMMER includes the following sliding block models:

  • Rigid (i.e., traditional Newmark analysis (Newmark 1965) as implemented by R. W. Jibson and Keefer (1993)).

  • Decoupled (per Makdisi and Seed (1978))

  • Coupled (described by Chopra and Zhang (1991) and modified by Rathje and Bray (1999))
